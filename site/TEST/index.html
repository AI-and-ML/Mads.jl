Testing & Verification

Testing & Verification

MADS includes a series of test and verification problems. These are also provided as examples how to run the code and perform various types of data and model analyses. All the MADS examples can be found in the MADS repository: github.

All the test and verification problems can be executed using:

import Mads
Mads.test()

or

Pkg.test("Mads")
Pkg.test("Anasol")
Pkg.test("BIGUQ")
Pkg.test("ReusableFunctions")
Pkg.test("MetaProgTools")
Pkg.test("RobustPmap")
